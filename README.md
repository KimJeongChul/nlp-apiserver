# nlp-apiserver

Example Golang NLP API server 
 - Supported Rest API

##
| API                             | PATH          | Method          |
|---------------------------------|---------------|-----------------|
| Create Train ID | /train    | POST     |
| Delete Train ID | /train/{trainID}/delete | POST |
